# 规范
按照模块路由创建文件夹
apis目录下文件为共享模块 
```
  例如： 系统设置-部门管理 API
  
  |-apis
| | | dictionary.js
  | |-system
  |   |-department
  |   |   index.js
  |-assets
  |-components
  |-directive
  |-filters
  |-icons
  |-layout
  |-router
  |-store
  |-styles
  |-utils
  |-vendor
  |-views


```
